## bitHuman Cloud (managed)

No local runtime — call the REST API or use the LiveKit plugin.

| Example | Model | What it does | Source |
|---------|-------|--------------|--------|
| **Agent Management** | — | Validate credentials, get/update agents | [api/](https://github.com/bithuman-product/bithuman-examples/tree/main/api) |
| **Agent Generation** | — | Create agents from prompt, poll status | [api/](https://github.com/bithuman-product/bithuman-examples/tree/main/api) |
| **Dynamics** | Essence | Generate gestures, list available gestures | [api/](https://github.com/bithuman-product/bithuman-examples/tree/main/api) |
| **Essence + Cloud Plugin** | Essence (CPU) | LiveKit avatar with hosted Essence agent | [essence-cloud/](https://github.com/bithuman-product/bithuman-examples/tree/main/essence-cloud) |
| **Expression + Cloud Plugin** | Expression (GPU) | LiveKit avatar with hosted Expression agent or any face image | [expression-cloud/](https://github.com/bithuman-product/bithuman-examples/tree/main/expression-cloud) |

## On-device Apple Silicon

Everything runs on the user's Mac, iPad, or iPhone — only the billing heartbeat goes out.

| Example | Surface | What it does | Source |
|---------|---------|--------------|--------|
| **[Expression on macOS (on-device)](/examples/apple-expression)** | Swift SDK | Floating avatar window, drag-drop face swap, Sparkle DMG | [bithuman-apps/Mac](https://github.com/bithuman-product/bithuman-apps/tree/main/Mac) |
| **iPad reference app** | Swift SDK | Stage Manager 320×320 panel, Picture-in-Picture, PhotosPicker portrait swap | [bithuman-apps/iPad](https://github.com/bithuman-product/bithuman-apps/tree/main/iPad) |
| **iPhone reference app** | Swift SDK | Full-screen avatar, tap-to-collapse 120 pt PiP | [bithuman-apps/iPhone](https://github.com/bithuman-product/bithuman-apps/tree/main/iPhone) |
| **[Apple Local Agent](/examples/apple-local)** | Python | 100% offline on macOS (Siri + Ollama, Essence) | [integrations/macos-offline/](https://github.com/bithuman-product/bithuman-examples/tree/main/integrations/macos-offline) |
| **[bitHuman Halo](/examples/halo-macos)** | Signed DMG | Free macOS desktop app — floating avatar, local STT + LLM + TTS | [Install guide](/examples/halo-macos) |
| **bithuman-cli** | CLI (Homebrew) | `brew install bithuman-cli` — voice / video / text modes | [Sources/BithumanCLI](https://github.com/bithuman-product/bithuman-kit/tree/main/Sources/BithumanCLI) |

Swift SDK overview: **[/swift-sdk/overview](/swift-sdk/overview)** · Quickstart: **[/swift-sdk/quickstart](/swift-sdk/quickstart)** · CLI reference: **[/swift-sdk/cli](/swift-sdk/cli)**.
