## Quick start

```bash
curl -X POST https://api.bithuman.ai/v1/tts \
  -H "api-secret: $BITHUMAN_API_SECRET" \
  -H "Content-Type: application/json" \
  -d '{"text": "Hello from bitHuman."}' \
  --output hello.wav
```

Returns 44.1 kHz, 16-bit mono WAV. Median first-byte ~120 ms.

## Streaming response format

When `stream: true` on `POST /v1/tts`, the body is a sequence of length-prefixed PCM frames:

```
[uint32 BE: frame bytes] [PCM int16 LE mono 44.1kHz]
[uint32 BE: frame bytes] [PCM ...]
...
[uint32 BE: 0]   ← terminator
```

`Content-Type: audio/pcm; rate=44100; framing=len32be`. First-byte ≈ 22 ms server-side.
