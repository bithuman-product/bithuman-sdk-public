bitHuman puts a real-time, lip-synced talking avatar in your app. Send audio in, get animated video out at 25 FPS with sub-200ms latency.

It runs **anywhere your users are** — iPhone, iPad, Android, Mac, Linux server, Raspberry Pi, or via a cloud REST API. Inference happens on the device. No cloud round-trip for rendering, no PII leaves the box, no dependency on network reachability for the avatar itself.

## Two avatars to choose from

| Avatar | What it is | Best for |
|---|---|---|
| **[Essence](/getting-started/models#essence)** | A pre-built avatar identity packaged in an `.imx` file. Real-time lip-sync, low memory, runs on every supported platform. | Kiosks, mobile apps, edge devices, high-concurrency servers. The default. |
| **[Expression](/getting-started/models#expression)** | A dynamic avatar that animates *any* portrait image at runtime. Higher quality close-up faces; needs more compute. | Native desktop apps, custom faces per session, cloud workloads with GPUs. |

Both models speak the same `.imx` file format and the same SDK methods — you can swap them without rewriting your integration.

## 5-minute quickstart

Pick the surface that matches what you're building:

<CodeGroup>

```sh CLI (no code)
brew install bithuman-product/bithuman/bithuman   # macOS

# Auto-picks cloud (OpenAI) when OPENAI_API_KEY is set, on-device otherwise.
bithuman voice                # spoken conversation in your terminal
bithuman text                 # text-only chat
bithuman avatar               # opens http://127.0.0.1:8080 with the avatar
```

```python Python (Linux / macOS / Windows)
# pip install bithuman
import asyncio, os
from bithuman import AsyncBithuman
from bithuman.audio import float32_to_int16, load_audio

async def main():
    runtime = await AsyncBithuman.create(
        model_path="avatar.imx",
        api_secret=os.environ["BITHUMAN_API_SECRET"],
    )
    await runtime.start()

    pcm, sr = load_audio("speech.wav")
    pcm = float32_to_int16(pcm)
    chunk = sr // 100                            # 10 ms per chunk
    for i in range(0, len(pcm), chunk):
        await runtime.push_audio(
            pcm[i : i + chunk].tobytes(), sr, last_chunk=False,
        )
    await runtime.flush()

    async for frame in runtime.run():
        if frame.has_image:
            image = frame.bgr_image              # numpy (H, W, 3) uint8
        if frame.end_of_speech:
            break
    await runtime.stop()

asyncio.run(main())
# Full guide: /getting-started/quickstart
```

```bash REST API (any language, cloud)
# 1. Verify your API secret.
curl -X POST https://api.bithuman.ai/v1/validate \
  -H "api-secret: $BITHUMAN_API_SECRET"

# 2. Make a hosted agent speak. Replace AGENT_CODE with your own
#    (create one at https://www.bithuman.ai/#explore).
curl -X POST https://api.bithuman.ai/v1/agent/AGENT_CODE/speak \
  -H "api-secret: $BITHUMAN_API_SECRET" \
  -H "content-type: application/json" \
  -d '{"message": "Hello from the REST API."}'
# Full reference: /api-reference/overview
```

## Platform support

bitHuman runs on every major platform. The runtime is on-device by default; only a lightweight billing heartbeat (one request per minute) touches the network.

| Platform | Avatars | Minimum hardware |
|---|---|---|
| **iOS** | Essence | iPhone 17 Pro+ |
| **iPadOS** | Essence + Expression | iPad Pro M4+ |
| **macOS Apple Silicon** | Essence + Expression | M3+ for Expression; any Apple Silicon for Essence |
| **macOS Intel** | Essence | Any Intel Mac |
| **Android** | Essence | `arm64-v8a`, Android 10+ |
| **Linux x86_64** | Essence | Any CPU, 4 GB RAM |
| **Linux ARM64** | Essence | Raspberry Pi 4B+ |
| **Windows x86_64** | Essence | Any CPU, 4 GB RAM |
| **NVIDIA GPU (Docker)** | Expression | 8 GB+ VRAM |
| **bitHuman Cloud** | Essence + Expression | API key only |
