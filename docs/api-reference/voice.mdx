## Endpoints

<AccordionGroup>

<Accordion title="POST /v1/tts — Synthesize speech" icon="play">

Generate audio (WAV or PCM) from a text prompt. Default is non-streaming WAV; pass `stream: true` for sentence-chunked PCM frames.

<ParamField body="text" type="string" required>
  Text to synthesize. Any length, multi-sentence supported.
</ParamField>
<ParamField body="voice" type="string" default="M1">
  Voice ID. Built-in: `M1`-`M5` (male), `F1`-`F5` (female).
</ParamField>
<ParamField body="language" type="string" default="en">
  ISO-2 language code. 31 languages supported (see catalog below).
</ParamField>
<ParamField body="total_steps" type="integer" default="8">
  Diffusion quality. `5`=fast/lower, `8`=balanced, `12`=highest. Range 1-100.
</ParamField>
<ParamField body="speed" type="number" default="1.05">
  Playback rate. Range `0.7`-`2.0`.
</ParamField>
<ParamField body="format" type="string" default="wav">
  Response format: `wav` or `pcm_s16le`.
</ParamField>
<ParamField body="stream" type="boolean" default="false">
  If true, returns sentence-chunked length-prefixed PCM frames.
</ParamField>

<RequestExample>
```bash curl
curl -X POST https://api.bithuman.ai/v1/tts \
  -H "api-secret: $BITHUMAN_API_SECRET" \
  -H "Content-Type: application/json" \
  -d '{"text": "Hello from bitHuman.", "voice": "F1"}' \
  --output hello.wav
```

```python python
import requests
r = requests.post(
    "https://api.bithuman.ai/v1/tts",
    headers={"api-secret": "YOUR_API_SECRET"},
    json={"text": "Hello from bitHuman.", "voice": "F1"},
)
open("hello.wav", "wb").write(r.content)
```

```javascript node
const res = await fetch("https://api.bithuman.ai/v1/tts", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "api-secret": process.env.BITHUMAN_API_SECRET,
  },
  body: JSON.stringify({ text: "Hello from bitHuman.", voice: "F1" }),
});
require("fs").writeFileSync("hello.wav", Buffer.from(await res.arrayBuffer()));
```
</RequestExample>

<ResponseExample>
```
HTTP/1.1 200 OK
Content-Type: audio/wav
Content-Length: 145980

<binary WAV bytes>
```
</ResponseExample>

**Streaming framing** (`stream: true`): `[uint32 BE size][PCM int16 LE mono 44.1kHz]` per sentence, terminator `[uint32 BE = 0]`. `Content-Type: audio/pcm; rate=44100; framing=len32be`. First-byte ≈ 22 ms server-side.

</Accordion>

<Accordion title="POST /v1/audio/speech — OpenAI-compatible drop-in" icon="rotate">

Drop-in compatible with OpenAI's [`POST /v1/audio/speech`](https://platform.openai.com/docs/api-reference/audio/createSpeech). Swap the base URL to `https://api.bithuman.ai/v1` and the auth header to `api-secret`.

<ParamField body="model" type="string">
  Ignored — bitHuman uses its own model. Pass `"tts-1"` for client compatibility.
</ParamField>
<ParamField body="input" type="string" required>
  Text to synthesize.
</ParamField>
<ParamField body="voice" type="string" default="M1">
  bitHuman voice ID (`M1`-`F5`).
</ParamField>
<ParamField body="response_format" type="string" default="wav" />
<ParamField body="speed" type="number" default="1.05" />

<RequestExample>
```python openai-client
from openai import OpenAI
client = OpenAI(
  base_url="https://api.bithuman.ai/v1",
  api_key="not-used",
  default_headers={"api-secret": "YOUR_API_SECRET"},
)
audio = client.audio.speech.create(
  model="tts-1",
  input="Hello, served as if it were OpenAI.",
  voice="F1",
)
audio.stream_to_file("hello.wav")
```
</RequestExample>

</Accordion>

<Accordion title="GET /v1/studio/axes — List voice-tuning axes" icon="sliders">

<Accordion title="POST /v1/studio/preview — Preview a tuned voice" icon="wand-magic-sparkles">

Synth a one-off clip with an axis-edited voice. The edited voice is ephemeral (not persisted to the catalog).

<ParamField body="text" type="string" required />
<ParamField body="base_voice" type="string" required>
  Built-in voice ID (`M1`-`F5`) to edit.
</ParamField>
<ParamField body="axes" type="object" required>
  Map of axis-name → float value, e.g. `{"pitch": 0.15, "brightness": -0.10}`.
</ParamField>
<ParamField body="language" type="string" default="en" />

<RequestExample>
```bash curl
curl -X POST https://api.bithuman.ai/v1/studio/preview \
  -H "api-secret: $BITHUMAN_API_SECRET" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "Tuned voice preview.",
    "base_voice": "F1",
    "axes": {"pitch": 0.15, "brightness": -0.10}
  }' \
  --output tuned.wav
```
</RequestExample>

<Tip>
The **Tune** popover at [www.bithuman.ai/voice](https://www.bithuman.ai/voice) calls these two endpoints — drag the sliders for live previews.
</Tip>

## Quick start

```bash
curl -X POST https://api.bithuman.ai/v1/tts \
  -H "api-secret: $BITHUMAN_API_SECRET" \
  -H "Content-Type: application/json" \
  -d '{"text": "Hello from bitHuman."}' \
  --output hello.wav
```

Returns 44.1 kHz, 16-bit mono WAV. Median first-byte ~120 ms.

## Errors

| Status | Cause |
|--------|-------|
| `400` | Missing `text`, unsupported voice/language, out-of-range params |
| `401` | Missing or invalid `api-secret` |
| `429` | Rate limit exceeded |
| `503` | Queue full; retry with backoff |
