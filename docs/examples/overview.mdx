## On-device — both Essence and Expression

Everything runs on the user's Mac, iPad, iPhone, or Android device — only the billing heartbeat goes out. Both model families have first-class example apps.

### Expression on-device (Apple Silicon)

| Example | Surface | What it does | Source |
|---------|---------|--------------|--------|
| **[Expression on macOS (Python SDK)](/examples/apple-expression)** | Python SDK | `pip install bithuman` + bundled `bithuman-expression-daemon`. Render any portrait + WAV → MP4. | [integrations/offline-mac/](https://github.com/bithuman-product/bithuman-sdk-public/tree/main/Examples/integrations/offline-mac) |
| **Mac reference app** | Native Swift app | Floating avatar window, drag-drop face swap, on-device LLM/TTS via MLX, Sparkle DMG | [`bithuman-apps/expression/mac/`](https://github.com/bithuman-product/bithuman-apps/tree/main/expression/mac) |
| **iPad reference app** | Native Swift app | Stage Manager 320×320 panel, Picture-in-Picture, PhotosPicker portrait swap | [`bithuman-apps/expression/ipad/`](https://github.com/bithuman-product/bithuman-apps/tree/main/expression/ipad) |
| **iPhone reference app** | Native Swift app | Parked in `archive/` while the per-app memory budget firms up — iPhone today is Essence-only on-device. | `bithuman-apps/archive/iPhone` |

### Essence on-device (mac · iOS · iPadOS · Android)

| Example | Surface | What it does | Source |
